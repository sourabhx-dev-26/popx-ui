
# PopX UI – React Assignment

A mobile-friendly React application replicating PopX's registration and login experience.  
This project was built as an assignment to practice modern React, clean component structures, and responsive design.

🔗 [Live Demo](https://popx-ui-six.vercel.app/)

---

## 🚀 Features

- User registration and login pages
- Mobile-first, responsive layout
- Smooth navigation with React Router
- Simple, clean, reusable React components
- Minimal, readable CSS styling

## 🛠️ Tech Stack

- [React](https://react.dev/)
- [Vite](https://vitejs.dev/)
- CSS

## 📦 How to Run Locally

```bash
git clone https://github.com/sourabhx-dev-26/popx-ui.git
cd popx-ui
npm install
npm run dev
```
Open [http://localhost:5173](http://localhost:5173)  
(Port may vary, see terminal after running above command.)

---

## 📁 Folder Structure

```
src/
  components/      # Reusable UI components (Button, Input, PhoneFrame, etc.)
  pages/           # Main app pages (Login, Signup, Profile, Landing)
  styles/          # Global and component styles
  main.jsx         # App entry point
```

---

## 🎯 Objective / Assignment

Recreate the PopX registration and login flow in React, focusing on:
- Clean and reusable component structure
- Mobile-first design principles
- Seamless routing between screens
